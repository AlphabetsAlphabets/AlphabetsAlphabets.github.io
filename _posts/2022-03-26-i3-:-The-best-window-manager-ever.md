# i3: The best window manager ever.
I've recently switched from Awesome to i3. Why? Because, my entire system broke. I'm not sure what's going on, but I was unable to even get to the login screen before I saw red text during startup. So, I had to wipe my disk clean and put on a fresh install of Manjaro. A friend of mine suggested that I switch to i3. Ever since switching to i3, I've only had good things to say about it.

> "Awesome does too many things at once. That's why it sucks." - Bare

Bare is the one who recommended me to switch to i3, he also helped me a lot to recover my data (thanks again). i3 does not do too many things at once, i3 is a *window manager* so all it does it manage windows. i3's default bar is a seperate executable, that's distinct from the window manager itself. The config file is easy to read because it isn't code, plus the user guide is fantastic. There I learned about i3's best feature - containers.

You can have containers within containers, basically a container is a collection of windows. In a container it can be in one of three modes vertical, horizontal or tabbed. The first two are obvious, and tabbed is where everything in that container will take up the maximum space of that container (like having multiple windows be maximized). So, you can have a container with documentation open and another container with neovim.

![Standard workflow]

Keep in mind that the container on the right is independent from the one on the left. With rofi I can easily switch between windows if it gets to cluttered. Being able to manage the windows myself instead of needing to choose from a list of default presets is amazing. I believe Awesome will restrict you to 10 or so different tiling presets, while i3 has none so I can handle everything myself. This makes it so I can study extremely easily whether it be for school or something more personal like Japanese.
