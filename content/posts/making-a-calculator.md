# Operator precedence
A calculator relies on something called operator precedence as without the the entire thing breaks. What is it? Operator precedence defines the correct order of operations. You perform muliplication before addition or subtraction, you perform exponents before multiplication, stuff like that.

Implementing this took a while and the source code can be found in the [project repository](https://github.com/AlphabetsAlphabets/Kalculator). This is still quite incomplete as it doesn't handle brackets and exponents at the same time. Currently this is on hold because I got bored of it and wanted to do other things.
